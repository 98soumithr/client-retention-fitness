Demo Script for Presenters

  1. Start on the Dashboard tab — show the overview stats and at-risk clients
  2. Switch to Clients — show the full client list with status badges
  3. Come to Demo Controls — click "Advance +3 Days" to simulate time passing
  4. Click "Run Retention Check" — the engine will detect at-risk clients
  5. Switch to Follow-ups — show the emails that were automatically sent
  6. Click "View Email Previews" to show the actual emails that were generated
  7. Advance more days and repeat to show the escalation flow
  8. Use "Reset Demo" to start fresh if needed
